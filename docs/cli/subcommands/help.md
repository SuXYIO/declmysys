# Subcommand-Help

Gives you the commandline help.

Can't spec which subcmd to show help, so `-h` and `--help` is more recommended.

## Args

```
declmysys help [SUBCOMMAND]
```

- `SUBCOMMAND`: Specify the subcommand to show help to, if not specified or cannot find subcommand with this name, shows help for the main program

## Aliases

Similar to `--help` or `-h`
